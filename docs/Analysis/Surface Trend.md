# 曲面走势分析能力

通过光影来判断曲面走势，这个一直是做交通工具设计中调整曲面和造型的一种方法。设计师需要对这个有所了解，每一个微妙的曲面变化在光影下都会有变化。了解了这个你画手绘的时候就知道如何处理复杂曲面了。
